up.)  Serves as a Toodledo client.


0. HOW TO BUILD
=================

[Fire emulator/device up...]

$ ant clean debug uninstall install run
...
BUILD SUCCESSFUL
Total time: xx seconds

=============

 * Can add task
 * Can set userid/password
